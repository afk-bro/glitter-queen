### Task 4: AnnouncementBar → gold

**Files:**
- Modify: `components/layout/AnnouncementBar.tsx`
- Test: `__tests__/components/AnnouncementBar.test.tsx`

**Why `text-[#1a1a1a]`:** `--accent-foreground` is `#1a1a1a` in light mode but flips to `#f5f5f5` in dark mode. `#f5f5f5` on `#d4af37` (dark-mode gold) is only ~1.9:1 — WCAG fail. Hardcoding dark text forces correct contrast in both modes without modifying the global token.

- [ ] **Step 1: Update the tests (write failing assertions)**

In `__tests__/components/AnnouncementBar.test.tsx`, replace the two existing bg assertions:

From:
```tsx
  it('has bg-primary class on root element', () => {
    const { container } = render(<AnnouncementBar message="Test" />)
    expect(container.firstChild).toHaveClass('bg-primary')
  })

  it('does not have bg-primary-soft class', () => {
    const { container } = render(<AnnouncementBar message="Test" />)
    expect(container.firstChild).not.toHaveClass('bg-primary-soft')
  })
```

To:
```tsx
  it('has bg-accent class on root element', () => {
    const { container } = render(<AnnouncementBar message="Test" />)
    expect(container.firstChild).toHaveClass('bg-accent')
  })

  it('does not have bg-primary class', () => {
    const { container } = render(<AnnouncementBar message="Test" />)
    expect(container.firstChild).not.toHaveClass('bg-primary')
  })
```

- [ ] **Step 2: Run test to verify it fails**

```bash
npx vitest run __tests__/components/AnnouncementBar.test.tsx
```

Expected: FAIL — `has bg-accent class on root element` fails (element has `bg-primary`); `does not have bg-primary class` fails for the same reason.

- [ ] **Step 3: Update the component**

In `components/layout/AnnouncementBar.tsx`, change the root `<div>` opening tag and the dismiss button class:

From:
```tsx
      <div className="relative bg-primary text-primary-foreground text-center py-2 px-10 text-sm font-body tracking-wide">

        {dismissible && (
          <button

          >
```

To:
```tsx
      <div className="relative bg-accent text-[#1a1a1a] text-center py-2 px-10 text-sm font-body tracking-wide">
        <span>{message}</span>

          <button
            onClick={() => setDismissed(true)}
            className="absolute right-3 top-1/2 -translate-y-1/2 text-[#1a1a1a] opacity-80 hover:opacity-60 transition-opacity duration-200"
```
